Estruturas de Controle

1. Condicionais

As estruturas condicionais permitem que o programa tome decisões com base em condições específicas. Elas são fundamentais para tornar os programas dinâmicos e adaptáveis.

Definição de Condicionais

As estruturas condicionais avaliam expressões booleanas e, dependendo do resultado, executam diferentes blocos de código. Em C, os principais tipos de estruturas condicionais são:

  • if: Executa um bloco de código se a condição for verdadeira.
  • if...else: Executa um bloco de código se a condição for verdadeira e outro se for falsa.
  • if...else if: Permite testar várias condições em sequência.

Exemplos de Condicionais

#include <stdio.h>
int main() {
    int idade = 20;
    if (idade >= 18) {
        printf("Você é maior de idade.\n");
    }
    return 0;
}
            
#include <stdio.h>
int main() {
    int nota = 75;
    if (nota >= 60) {
        printf("Você foi aprovado!\n");
    } else {
        printf("Você foi reprovado!\n");
    }
    return 0;
}
            
#include <stdio.h>
int main() {
    int hora = 14;
    if (hora < 12) {
        printf("Bom dia!\n");
    } else if (hora < 18) {
        printf("Boa tarde!\n");
    } else {
        printf("Boa noite!\n");
    }
    return 0;
}

2. Laços de Repetição

Os laços de repetição permitem executar um bloco de código repetidamente enquanto uma determinada condição é verdadeira. Os principais tipos de laços em C são:

  • for: Um laço controlado com um número fixo de iterações.
  • while: Executa o bloco de código enquanto a condição for verdadeira.
  • do...while: Executa o bloco de código pelo menos uma vez, verificando a condição no final.

Exemplos de Laços

#include <stdio.h>
int main() {
    for (int i = 0; i < 5; i++) {
        printf("Iteração %d\n", i);
    }
    return 0;
}
            
#include <stdio.h>
int main() {
    int i = 0;
    while (i < 5) {
        printf("Valor de i: %d\n", i);
        i++;
    }
    return 0;
}
            
#include <stdio.h>
int main() {
    int i = 0;
    do {
        printf("Valor de i: %d\n", i);
        i++;
    } while (i < 5);
    return 0;
}

3. Exemplos Práticos

Exemplo 1: Calculando a soma de números pares

#include <stdio.h>
int main() {
    int soma = 0;
    for (int i = 1; i <= 10; i++) {
        if (i % 2 == 0) {
            soma += i;
        }
    }
    printf("A soma dos números pares de 1 a 10 é: %d\n", soma);
    return 0;
}
