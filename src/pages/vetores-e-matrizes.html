Vetores e Matrizes

1. Definição de Vetor e Matriz

Vetor

Um vetor é uma estrutura de dados que armazena múltiplos valores do mesmo tipo de forma sequencial na memória. Os elementos são acessados por índices, começando de 0.

#include <stdio.h>
int main() {
    int idades[5] = {18, 22, 25, 30, 28};
    // Acessando e exibindo os elementos do vetor
    printf("Idade da primeira pessoa: %d\n", idades[0]);
    printf("Idade da segunda pessoa: %d\n", idades[1]);
    return 0;
}

Matriz

Uma matriz é uma estrutura de dados bidimensional, organizada em linhas e colunas. Cada elemento é acessado por dois índices: linha e coluna.

#include <stdio.h>
int main() {
    int tabela[2][3] = {{1, 2, 3}, {4, 5, 6}};
    // Acessando e exibindo os elementos da matriz
    printf("Elemento na posição [0][0]: %d\n", tabela[0][0]);
    printf("Elemento na posição [1][2]: %d\n", tabela[1][2]);
    return 0;
}

2. Funcionamento de Vetores e Matrizes

Funcionamento de Vetores

Os vetores são armazenados de forma sequencial na memória. Eles permitem acesso rápido e eficiente a cada elemento utilizando índices.

Funcionamento de Matrizes

As matrizes são organizadas em duas dimensões (linhas e colunas) e são úteis para armazenar dados tabulares ou bidimensionais. Cada elemento é acessado utilizando dois índices.

3. Exemplos dos Conceitos

Exemplo de Vetor

Um vetor em C pode ser declarado e inicializado como mostrado abaixo:

#include <stdio.h>
int main() {
    int idades[5] = {18, 22, 25, 30, 28};
    printf("Idade da terceira pessoa: %d\n", idades[2]);
    return 0;
}

Exemplo de Matriz

Uma matriz bidimensional pode ser usada para armazenar tabelas de dados:

#include <stdio.h>
int main() {
    int tabela[3][3] = {{1, 2, 3}, {4, 5, 6}, {7, 8, 9}};
    printf("Elemento na posição [2][1]: %d\n", tabela[2][1]);
    return 0;
}

4. Importância e Aplicações de Vetores e Matrizes

Vetores

Vetores são amplamente utilizados para armazenar dados sequenciais, como listas de números ou nomes. Eles são essenciais para tarefas como ordenação, busca e manipulação de dados.

Matrizes

Matrizes são ideais para organizar dados em tabelas ou realizar cálculos matemáticos, como multiplicação de matrizes ou transformações geométricas.
